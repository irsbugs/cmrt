# Server for Wordpress and Standalone CiviCRM

2026-05-13
Ian

## Objective

VentraIP host the [cmrailtrial.org.au](cmrailtrial.org.au) Wordpress website, while CiviCRM is hosted on cmrailtr.civicrm.org. The intention is to have CiviCRM Standalone installed on VentraIP hosting system. The Wordpress website would be the main domain and the CiviCRM would be the sub-domain crm.cmrailtrail.org.au.
