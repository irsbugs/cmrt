# CMRT ACL for Email links to a Profile.md

2026-04-30 - Ian Stewart

Objective: Have CMRT Mailing send emails to each contact in group within CMRT (E.g. "Current Members"). Each email contains a link to click on. Each link is a unique URL and is only for the contact that received the email. When Spark Essentials recieves the URL request it will open up a Profile ("questionaire") for the contact in the group to fill in.
