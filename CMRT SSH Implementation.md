# SSH Implementation

## Introduction

SSH, *Secure SHell*, access into the VentraIP *cmrailtr* account allows a remote PC to issue bash commands to run applications. For example, backing up the sql database. It provides a similar service as C-Panel *Terminal*.

SCP, *Secure CoPy*, is a partner application to *SSH*, which allows the copying of files from a remote PC to the VentraIP *cmrailtr* account and vice versa.
